Getting Started

The School Management System API provides programmatic access to school data. All API endpoints return JSON data.

Authentication

Currently, the API does not require authentication for development purposes. In a production environment, proper authentication would be implemented.

Base URL

http://localhost:5000/api

Common Response Codes

  • 200 OK Success
  • 201 Created Resource Created
  • 400 Bad Request Invalid Input
  • 404 Not Found Resource Not Found
  • 500 Internal Server Error Server Error
